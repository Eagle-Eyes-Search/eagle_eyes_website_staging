System Requirements

Eagle Eyes Pilot
Operating System Android 7+ (Does not work on Google Pixel 6/7 due to DJI SDK bug)
Processor Snapdragon 855 or better (Samsung Galaxy S10 equivalent or faster)
RAM 4GB minimum, 6GB+ recommended
Storage At least 2GB free space for maps and detections
Drone Compatibility Supports select DJI drones using DJI SDK 4 & 5 (e.g., Mini 2, Mini 3 Pro, Mavic 3T, Matrice 30T)
Controller Compatibility Works with RC-N1, RC-Pro, RC-Plus; Not compatible with DJI Smart Controller or RC-2
Eagle Eyes Scan
Operating System Windows 10/11, macOS Ventura (13) or Sonoma (14)
Processor Any modern Intel Core i3/i5/i7 or AMD Ryzen (2013+); Not supported: Celeron, Atom, or Pentium Silver
RAM 8GB minimum, 16GB recommended
GPU Not required (currently unused), but may be supported in future updates
Storage At least 2GB of free space, more recommended for video storage
Drone Compatibility Works with any drone that records video; optimized for DJI drones (supports DJI file naming & GPS data)
Live Video Input Requires HDMI capture card for live video from a drone controller OR RTMP network streaming
