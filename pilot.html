Eagle Eyes Pilot

An Android App with a built-in computer vision system, for piloting DJI drones.

The system scans the live video stream from the drone for color and motion anomalies, bringing any detections to the attention of the operator, while recording their coordinates. Eagle Eyes Pilot also enables live tracking of the drone in your team’s CalTopo map and allows the operator to add full-resolution, geolocated drone images directly into the map.

Find more info in the Eagle Eyes Pilot User Manual.

Compatibility

What's been tested

Eagle Eyes Pilot officially supports the following drones/controllers:

  • Mavic 3T with RC-Pro controller
  • Mini 3 Pro with RC-Pro Controller and RC-N1 controller

Early compatibility - works but needs more field testing:

  • Matrice 30T with RC-Plus Controller

External users have reported that Eagle Eyes Pilot also works with:

  • Anzu Raptor T

We expect to add support for the following drones soon:

  • Mavic 2 Enterprise Dual with RC-1A controller
  • Mini 2 with RC-N1 controller
  • Air 2S with RC-N1 controller

Note: So far, we’ve focused on DJI compatibility for Eagle Eyes Pilot. However, we’re actively working toward broader support for other platforms like Skydio and Autel as we move toward making Eagle Eyes a truly drone-agnostic solution. In the meantime, our desktop software, Eagle Eyes Scan, can run the detector on any clean HDMI stream or on stored images and video.
